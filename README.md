**slurricane.dev** is a feature-rich script for *South London Remastered* on Roblox, designed to automate the grind and let you focus on what matters. Clean UI, smooth execution, no fluff.

### 💰 Money Farms

#### 🧠 Auto IDEA Job
> Farms you upwards of **£100,000 in 20–30 minutes** at max job level.

- Works best at **max job level**
- Consistent and reliable income

- Handles all spill detection and cleaning
- Low-effort passive grind

- ⚠️ **Must be in the Banklog bando area to work**

- Ideal for levelling up fast while making money

## 🚀 Getting Started

### Prerequisites
- A working Roblox executor (e.g. Potassium, Cosmic, etc.)

### Execution
```lua
loadstring(game:HttpGet("https://slurricane.dev/script.lua"))()
```

---

## 📖 Usage

1. Join **South London Remastered** on Roblox
2. Open your executor and attach it
3. Paste the loadstring above and execute
4. The UI will load — navigate the tabs to enable features
5. For **Auto Banklog**, make sure you're in the bando area before enabling
